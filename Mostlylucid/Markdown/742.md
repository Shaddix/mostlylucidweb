I have downloaded and installed an interesting application - [BlogJet](http://blogjet.com/). It's a Windows-client for my blog tool (as well as some for other tools).

Actually, it was *not* me who wrote this text: when I launched [BlogJet](http://blogjet.com/) for the first time, it's edit window already contained this text and the program asked me to click *Post and Publish* button to post this text to my blog in order to test the connection (yes, and the text above is not mine too! :-)... So, I did.

Now I'm gonna learn all that cool features of [BlogJet](http://blogjet.com/) (don't ask **me**, there are too much - go to [their website](http://blogjet.com/) and read).

I thought I'd post this - VERY nice tool - thanks to [Paschal](http://weblogs.asp.net/pleloup/)for [pointing this thing out](http://weblogs.asp.net/pleloup/archive/2004/01/19/60252.aspx), very nice, simple tool - way better than [w.bloggar](http://wbloggar.com/)!
