I was doing some work on the new version of my first article last night and I got to thinking, all the code in the new version is in the same font / colours that I use in my VS.NET. So, I was showing the new version to a friend and he mentioned that most people use fixed-width fonts when coding like [profont](http://www.tobias-jung.de/seekingprofont/). This got me to thinking, am I missing some trick here, I always use Verdana 8pt on a 1600x1200 120DPI screen with tuned Cleartype when I code - what's the disadvantages of this - why would I want to use a fixed-width font?
