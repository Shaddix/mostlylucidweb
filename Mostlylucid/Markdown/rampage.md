# Introduction
So, quick post to explain why I went nuts releasing stuff this week. Like any ancient developer, I have a whole archive of little projects I started, then gave up on, over the years. 
WELL, it turns out tools like [Claude Code](https://www.claude.com/product/claude-code) are like completion machines! 

# This Site
Oh, and hopefully this site has started getting a bit more stable. I've tweaked a BUNCH of aspects to make it load insanely fast (no images, limited external dependencies, tuned hosting, etc.). 
Unlike MOST, I made the crazy decision to both start blogging and build my dream blogging platform. 
These posts are just [markdown files](https://www.mostlylucid.net/api/raw/rampage?language=en) and I have a VERY clever system I built to process, translate, assemble from remote locations, display diagrams, highlight code, etc., etc.

It makes life interesting but, again, as with my AI tips below: JUST BUILD SHIT. It doesn't matter if the entire customer base is you; build it, release it on NuGet (or npm, Docker, etc.). To survive we ALL need to start building incredible stuff. 
YOU have creativity and an understanding of the world; the AI can just write code. 

# HUH?

Now, while sitting here on my recliner, I can get to those projects I was stuck on—the ones I couldn't be bothered writing docs for (I never release those; nothing more frustrating!), those crazy ideas I didn't have the energy to work on. 
So I'm completing those projects at pace now (I already have 3 more queued up for when I get back into it!). 

Be in no doubt, agents like Claude Code are a GAME CHANGER for developers. That's not all entirely positive. We can get a TON more done, but MANY of us will be replaced by these things. 

I started using AI for coding as soon as ChatGPT got good enough to start making HTML. I rarely have the patience to make Razor Views/HTML pages, so I'd make one, paste it into the chat window (maybe with a ViewModel), and ask for a different version. GAME CHANGER.

Then, as they evolved, we got things like AI Chat in Rider, then Junie (more powerful, MCP-enabled so it could *do stuff*). At that point you could build whole features with a chat window; it was pretty good but still a bit slow and made mistakes. 

THEN Claude Code... and OH MY... it's wonderful. I've built entire platforms with Claude Code and it's a WHIZZ at unit testing. 
I urge you to try it; I BEG you to build lots of stuff in C# and make our ecosystem more vibrant. 

Just remember: just because you 'built it with AI', you're still responsible for what you produce. MAKE IT GOOD.


## My Tips for surviving the imminent AI-pocalypse
* Get GOOD at using them. I mean get really good—work out how to use them to build stuff. Work out what skills you need to succeed with them (hint: you still need to write specs—it's how the AI knows what to really do). 
* Understand what changes are upon us all—the code has never really been the hard part; now that's not really even a part at all much of the time. Focus on how you define features, what GREAT UX looks like, how to design a SYSTEM. Those skills are what will keep you relevant.
* Embrace the change; it's coming whether you like it or not. The sooner you get on board the better. Buy a Claude Code/Codex/Whatever subscription and start building. It's like when you learned to code. You need to learn, become proficient, then you can start to master it.
* Build stuff; don't just play. The more you build the better you get. The better you get the more you can do. The more you can do the more valuable you are.
* Network with other developers; share tips, tricks, and experiences. The community will be a huge help as we all navigate this new world together.

# Future?
I think we all have to realise that we're staring down a VERY different future, with the rise of AI tools, the imminent popping of the AI investment bubble, and a concurrent recession.

I really do fear for Junior Coders; I don't think they'll be NEEDED in future and that's devastating for the future of our profession. 

We're all doomed to end our days directing AI tools, building apps, and cleaning up after the disasters they cause... FUN TIMES AHEAD!
