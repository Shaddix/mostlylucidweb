While I'm procrastinating, I thought I'd mention a few of the controls I'm using in this proejct...first is [WebChart](http://www.carlosag.net/Tools/WebChart/Default.aspx), an excellent free WebCharting control, lets you do lovely graphs to impress your clients!
Next up is [Denis Bauer's](http://www.denisbauer.com/weblog/)[DynamicControlsPlaceholder](http://www.denisbauer.com/ASPNETControls/DynamicControlsPlaceholder.aspx) - which if you haven't used yet - you should. I mention this control a lot but is really is fantastic for ASP.NET with dynamic UserControls.
Next...[TaskScheduler](http://www.codeproject.com/csharp/TSNewLib.asp), a phenomenally useful C# wrapper for the [Windows TaskScheduler](http://www.microsoft.com/windows2000/techinfo/howitworks/management/task_scheduler.asp) - now you can run tasks whenever you want!
