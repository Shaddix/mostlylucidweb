@mostlylucid/mermaid-enhancements

Interactive pan/zoom, fullscreen, export, and theme switching for Mermaid.js

Features

  • Interactive Pan & Zoom - Navigate large diagrams with mouse wheel and drag
  • Touch Support - Full touch gesture support for mobile devices
  • Fullscreen Lightbox - View diagrams in immersive fullscreen mode with integrated toolbar
  • Export to PNG/SVG - Download diagrams in high quality
  • Auto Theme Switching - Automatically adapts to light/dark mode
  • Responsive Design - Works perfectly on mobile and desktop
  • Keyboard Shortcuts - ESC to close lightbox

Installation

npm install @mostlylucid/mermaid-enhancements

Usage

import { init } from '@mostlylucid/mermaid-enhancements';
import '@mostlylucid/mermaid-enhancements/styles.css';

// After Mermaid is loaded
await init();

Toolbar Customization

Global Configuration: These settings apply to all diagrams on the page.

import { configure, init } from '@mostlylucid/mermaid-enhancements';

// Hide entire toolbar
configure({
  controls: {
    showControls: false
  }
});

// Hide export buttons
configure({
  controls: {
    export: false
  }
});

// Control individual buttons
configure({
  controls: {
    fullscreen: true,
    zoomIn: true,
    zoomOut: true,
    reset: false,       // Hide reset button
    exportPng: true,
    exportSvg: false    // Hide SVG export
  }
});

await init();
