mostlylucid.Markdig.FetchExtension Demo

Fetch and embed remote Markdown at render time using <fetch> tags

Try it out

Edit the markdown below and click "Render Markdown" to see the FetchExtension in action.

  • Use <fetch markdownurl="URL" pollfrequency="24"/> to embed remote content
  • Add transformlinks="true" to rewrite relative links
  • Add showsummary="true" to display fetch metadata
  • Use <fetch-summary url="URL"/> to show metadata separately
  • Add disable="true" to prevent processing (useful for documentation)
